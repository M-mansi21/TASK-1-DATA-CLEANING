DATA CLEANING STEPS PERFORMED:

1.Quickly Autofit Rows and Columns
→ Select all (Ctrl + A) → Double-click on any column/row border

2.Formatting Long Values
→ Right click → Format Cells → Number → select “Text”

3.Remove Multiple Like BREAKS BETWEEN ROWS
->Use Find & Replace (Ctrl + H)
Find what: Ctrl + J → Replace with: Space or blank

4.Remove Blank Rows
Select data → Go to Home → Find & Select → Go To Special → Blanks → Right click → Delete → Entire Row

5. Text to Column
Select column → Data → Text to Columns → Choose Delimiter → Finish

  
6. Fill Blanks WITH NULL
→ Select data → Ctrl + H → Find what: (keep blank) → Replace with: NULL → Replace All

8.Magical Trim Formula  - REMOVE EXTRA SPACE
